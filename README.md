# hair style api  


## Running Locally

Make sure you have [Node.js](http://nodejs.org/)  installed.

```sh
git clone https://github.com/alvinMemphis/mimbo-backend # or clone your own fork
cd mimbo-backend
yarn install
yarn start
```

or using  NPM
```sh
npm install
npm start
```


Your app should now be running on [localhost:3000](http://localhost:3000/).
